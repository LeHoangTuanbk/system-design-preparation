# System Design

System design interview preparation slides using Marp.

## Setup

```bash
npm install
```

## Usage

### Preview single file in browser

```bash
npm run slides:preview slides/leetcode-system-design.md
```

### Export all slides to HTML

```bash
npm run slides:html
```

### Export all slides to PDF

```bash
npm run slides:pdf
```

### Watch mode (auto-rebuild on changes)

```bash
npm run slides:watch
```

Output files will be generated in `dist/` folder.

## VS Code

Install recommended extension **Marp for VS Code** to preview slides directly in editor.
